# Source Architecture

The skill should distinguish between durable lenses and live diagnostic evidence.

## Canonical Sources

Canonical sources are for durable lenses.

They include books, major essays, talks, reports, and theoretical frameworks that help the skill recognize recurring system mechanics.

Use canonical sources to ask better questions, form hypotheses, and explain why a pattern matters.

Do not use canonical sources as a substitute for evidence about the specific problem.

## Live Diagnostic Material

Messy live material is for the actual diagnosis.

It includes:

- Agency documents
- Meeting minutes
- Bills and statutes
- Budgets
- Procurement documents
- Environmental review records
- Litigation
- Local reporting
- Case studies
- Interviews and field notes
- Implementation data

Use live material to test whether a lens actually applies in this case.

## Operating Rule

Canonical sources provide the map. Live diagnostic material checks the terrain.

The memo should be explicit when a claim comes from a durable lens, when it comes from case-specific evidence, and when it is still an uncertainty.

## Two-Pass Use

Pass 1 may produce a provisional hypothesis memo without live diagnostic research.

Pass 2 should revise that memo using live diagnostic material. This is when causal weights become more defensible, failure modes become more concrete, and the action portfolio can move from plausible to evidence-backed.
