# Knowledge Library

This folder is for the historical, theoretical, and domain-specific material that should inform the Abundance Systems Memo skill over time.

The memo workflow should not become a book report. The library exists so the skill can apply the right lens when the diagnosis shows that lens matters.

## Working Rule

Use all major abundance lenses lightly in every run, then go deep only where the weighted causal model shows explanatory weight.

For example:

- If the blocker is procedural accretion, draw more heavily from *Public Citizens* and *Why Nothing Works*.
- If the blocker is implementation, procurement, or state capacity, draw more heavily from *Recoding America*.
- If the blocker is land use, housing supply, or permitting, draw more heavily from housing and land-use work.
- If the blocker is regional immobility or geographic opportunity, draw more heavily from *Stuck*.
- If the blocker is broad scarcity politics or anti-growth assumptions, draw more heavily from *Abundance*.

## Library Types

- `lenses`: reusable diagnostic frames from books, essays, reports, and talks.
- `cases`: concrete wins, failures, reforms, and cautionary examples.
- `mechanisms`: recurring system mechanics such as veto points, procurement failure, litigation risk, capture, fragmentation, or hollow compliance.
- `prompts`: reusable questions that help expose hidden structure.
